# photoravel-fe

포토래블 프론트엔드

## Convention

### Commit

- feat: 새로운 기능 추가
- fix: 에러 수정
- docs: 문서 수정
- style: 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
- refactor: 코드 리팩토링
- chore: 빌드 업무 수정, 패키지 매니저 수정
- comment: 주석 추가 및 변경
- remove: 파일, 폴더 삭제
- rename: 파일, 폴더명 변경

### Merge

- 2명의 코드 리뷰 후 approve 통과

### Branch

- feature/\*: 새로운 기능 추가
